## Installation

1. **Clone the repo**
   ```sh
   git clone https://github.com/omarmorad/flutterevision
   ```
2. **Install packages**
   ```sh
   flutter pub get
   ```
3. **Run the app**
   ```sh
   flutter run
   ```
